Objetivo general del módulo 3

Desarrollar en el perito valuador las capacidades necesarias para crear  herramientas personalizadas sin necesidad de ser un programador experto , mediante el uso de Vibe Coding y Power BI , a fin de optimizar procesos técnicos, estructurar información de valoración, analizar datos con mayor claridad y generar visualizaciones útiles para la toma de decisiones profesionales.

Temas del módulo

Semana 6: Introducción al Vibe Coding para Peritos 6 horas

Uso de IA, como GitHub Copilot, para generar código HTML y JavaScript aplicado a calculadoras de depreciación (Ross-Heidecke) o capitalización de rentas.

Semana 7: Dashboards de Mercado con IA 6 horas

Creación de aplicaciones web sencillas para visualizar mapas de calor de valores de mercado, con apoyo de Power BI.

Semanas 8 y 9: Integración API y Webhooks 12 horas

Conectar nuestras aplicaciones con bases de datos externas o servicios de mapas para ampliar la automatización y consulta de información valuatoria.

Semana 10: Proyecto integrador 6 horas

Aplicación práctica de los conocimientos del módulo en un ejercicio integrador orientado al desarrollo de herramientas y visualizaciones útiles para la valuación profesional.
